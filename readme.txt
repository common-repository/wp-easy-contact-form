Wp Easy Contact Form allows you to create contact forms that can be customized to satisfy all of your website contact needs. 



== Description ==



Wp Easy Contact Form is Easy to use and add forms to page or template. Fields are easy to add, remove, and re-order. The Wp Easy Contact Form will let the user send emails to a site's admin.

Use Shortcode

Easy to use Different page or post by shortcode "[wecf_contact_form_shortcode]".

Use This Code in Page Template <code> <?php echo do_shortcode('[wecf_contact_form_shortcode]'); ?></code>.


== Installation ==



    Installation 
    
    
    Go to Plugin Menu and Click the Add New button.
    
    Under Install Plugins, click the Upload link.
    
    Select the plugin zip file then click the Install Now button.
    
    Click the Activate Plugin link.

    
Manual Installation 

  
    Copy unzip folder and paste /wp-content/plugins/ directory.
    
    Go to your WordPress dashboard and navigate to Plugins -> Installed Plugins and Active plugin.
